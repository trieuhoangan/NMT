# NMT
Data is stor in folder data
  - train.vi : store vietnamese sentences for training
  - train.en : store english sentences for training
  - tree_train.txt : store the parsed label of vietnamese sentences for training 
  - valid.vi : store vietnamese sentences for validating
  - valid.en : store english sentences for validating
  - tree_valid.txt : store the parsed label of vietnamese sentences for validating
  - test.vi : store vietnamese sentences for testing
  - test.en : store english sentences for testing
  - tree_test.txt : store the parsed label of vietnamese sentences for testing

to start training: run file main.py by "python main.py"
